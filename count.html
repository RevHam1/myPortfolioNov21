What is React? What is JavaScript? What is an API? What is HTML? What is CSS? I love this project because its my first. This app was fun and challenging - implementing the Ticket Master API and figuring how to use javaScript for its count-down functionality. This application display the upcoming events for a city-location and the amount of time remaining, days, hours, mintues and seconds before the event begins. Hurry get your ticket before its too late...
