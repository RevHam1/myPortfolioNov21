About...

This project utilizes Python, Django, and PostgreSQL on the backend with React, Redux, and React-Bootstrap on the frontend. It also uses AWS, Heroku and Gunicorn for deployment. This project is an E-Commerce application with loads of functionalities and features.

FUNCTIONALITIES

Functionalities/Features

  • Full featured shopping cart
  • Product reviews and ratings
  • Top products carousel
  • Product pagination
  • Product search feature
  • User profile with orders
  • Admin product management
  • Admin user management
  • Admin Order details page
  • Mark orders as delivered option
  • Checkout process (shipping, payment method, etc)
  • PayPal / credit card integration
